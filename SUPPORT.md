# Support for TFGuard

We are here to help you secure your Terraform infrastructure!

## 🔍 Troubleshooting
Before opening a support request, please check:
- **Our Documentation**: [tfgaurd.com/docs](https://tfgaurd.com/docs)
- **API Status**: Check the `/api/health` endpoint.
- **Common Issues**: See the Troubleshooting section in our [README.md](README.md).

## 🚀 Ways to Get Support

### 1. GitHub Issues
For bug reports, feature requests, or technical questions related to the GitHub Action, please open an issue in this repository.

### 2. Email Support
For account-related issues, billing questions (Premium tier), or private inquiries, please email us at:
**support@tfgaurd.com**

### 3. Community Discussions
Join our community to discuss security best practices and rule suggestions.

## 🕒 Response Time
- **Premium Tier**: Priority support with a 24-hour response time guarantee.
- **Standard Tier**: Best-effort support within 2-3 business days.

## 🛡️ Security Vulnerabilities
If you discover a security vulnerability in TFGuard, please do **not** open a public issue. Instead, email us directly at security@tfgaurd.com.
